# PromptMaster Backend

Backend Python organizado em camadas para a API e futura migracao do CLI legado.

## Sprint 1

- bootstrap de configuracao centralizado
- clientes Gemini e Supabase lazy
- rotas `health`, `prompts`, `projects` e `personas`
- primeiros testes unitarios e de integracao

## Executar

```bash
pip install -r requirements.txt
python run.py
```
